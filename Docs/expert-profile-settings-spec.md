# 专家账号资料设置 Spec

> 版本：V1.0
> 日期：2026-07-14
> 实现边界：专家门户前端本地设置，不新增鉴权、后端接口或数据库表
> 当前状态：已在 WSL 实现并通过 Node.js 20.19.4 构建；浏览器人工交互验收待确认

## 1. 目标

在专家账号左侧菜单增加“专家设置”，让专家维护少量展示资料。保存后立即同步左侧账号区域，并在刷新或重新登录后恢复。

## 2. 入口与信息架构

专家菜单顺序：

1. 专家工作台
2. 全部案例
3. 检修知识库
4. 知识图谱
5. 专家设置

设置页包含：

- 基本资料：显示姓名、专业职称、所属机构或团队；
- 专业能力：擅长领域、审核设备范围；
- 联系信息：联系方式（选填）；
- 账号摘要：账号 `expert`、账号类型和当前状态，只读展示。

## 3. 字段与规则

| 字段 | 默认值 | 规则 |
| --- | --- | --- |
| 显示姓名 | 王海峰 | 必填，最多 20 个字符；采用演示用虚构身份 |
| 专业职称 | 高级工控设备检修工程师 | 必填，最多 30 个字符 |
| 所属机构/团队 | 国家管网集团山东德州分输站 · 设备技术组 | 必填，最多 40 个字符 |
| 擅长领域 | 工控机、站控系统、散热系统、工业通信 | 使用顿号或逗号分隔，最多 5 项 |
| 审核设备范围 | 全部工控设备 | 下拉选择 |
| 联系方式 | 站内分机 806 | 选填，最多 50 个字符 |

## 4. 交互

- 表单发生变化后显示“有未保存修改”；
- 保存前校验三个必填字段和专业标签数量；
- 保存成功后显示成功提示，左侧账号姓名和职称立即更新；
- “恢复默认”只修改当前表单，用户仍需点击保存；
- 专家侧“重置演示数据”入口放在账号摘要底部，不在左侧常驻账号区展示；
- 数据保存到浏览器 `localStorage`，键名为 `la-expert-profile-v2`；
- 读取到异常数据时回退默认资料，不影响专家门户其他功能。

## 5. 视觉规范

- 复用专家门户深蓝控制台、青蓝高亮、绿色成功状态和现有表单控件；
- 页面使用主表单与账号摘要双栏布局；
- 小屏时改为单栏，不新增弹窗或独立路由；
- 使用 Lucide 现有图标，不新增图片资源或依赖。

## 6. 非目标

- 不提供密码修改、头像上传、权限管理和账号停用；
- 不同步真实后端用户数据；
- 不修改案例审核、知识发布或知识图谱业务状态；
- 不替代后续正式账号中心。

## 7. 验收标准

1. 专家登录后能从左侧菜单进入“专家设置”；
2. 六项资料可编辑，必填项和最多 5 个标签规则生效；
3. 保存后左侧账号信息即时更新；
4. 刷新页面后资料仍然存在；
5. 恢复默认并保存后回到默认专家资料；
6. 工程师账号不显示专家设置；
7. Node.js `20.19.4` 下前端构建通过。
