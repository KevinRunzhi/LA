## 2. 官方赛题原文摘要

赛题名称：

> A1 - 基于多模态大模型技术的设备检修知识检索与作业系统

组类：

> A组（本科、研究生、高职）

出题企业：

> 龙芯中科技术股份有限公司

赛题简介：

本赛题聚焦工业、能源、制造等领域设备检修的核心痛点，以多模态大模型技术为核心支撑，引导参赛者开发一款兼具智能化、实用性与创新性的设备检修知识检索与作业辅助系统。赛题旨在推动大模型技术与行业场景深度融合，破解传统设备检修中知识分散、检索低效、作业不规范、经验传承难等问题，打造可落地、高价值的行业智能助手，助力行业数字化转型与效率升级。

官方业务场景：

在钢铁、汽车制造等行业，大型企业产线设备密集且联动性强，设备稳定运行直接决定产能与效益。企业在传统运维中存在检修知识分散于手册、日志与老员工经验中形成“数据孤岛”、新人上手慢、非计划停机单次损失超百万元等痛点，亟需智能工具破解经验传承难、故障响应慢的问题。

基于上述业务背景，聚焦制造业核心生产设备检修，参赛者可基于本地大模型或 API（如 Qwen、DeepSeek 等），开发适配现场检修作业的系统，整合多模态检修资源，辅助一线人员快速检索方案、诊断故障，实现知识数字化沉淀与高效复用。

## 3. 官方基本功能需求

1. 采用 B/S 架构开发，软件需部署在自主指令系统 LoongArch 架构 + 银河麒麟高级服务器版上运行。

   注意：不满足该要求视为 0 分。

2. 支持本地部署大模型服务或云端大模型服务，具备 PC Web 或 App 可视化界面。

3. 多模态知识检索功能：

   - 支持文本、故障图片、设备型号等多类型信息输入。
   - 实现精准语义检索与跨模态匹配。
   - 快速调取检修手册等资源。

4. 标准化作业指引功能：

   - 嵌入检修全流程。
   - 提供步骤化操作指引。
   - 提供合规校验提醒。
   - 支持根据设备类型与检修等级个性化推送流程。

5. 知识沉淀与更新功能：

   - 支持一线人员上传检修案例、经验总结等内容。
   - 经审核后纳入知识图谱。
   - 实现知识实时更新。
   - 支持手动标注与修正大模型输出结果。
   - 优化系统适配性。
