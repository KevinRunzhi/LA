## 三、赛题应对要点（结合老师意见）

1. **龙芯是硬门槛也是加分叙事**：不只"能跑"，要能出示"逐迭代部署记录"——第一天空壳上机、每迭代验证，答辩时这就是工程能力证据。
2. **演示双端形态**：龙芯上主展示菜单式后台管理系统（体现"完整系统"），平板展示检修向导（体现现场体验）；两端连同一后端。
3. **评分对位**：
   - 功能完整性 → 七模块功能分解树逐项对照演示；
   - 知识沉淀/人工修正 → 后台案例审核回流 + 金标命中桥段；
   - 创新性 → 三维定位、动态向导、语音伴随、离线内容生产线；
   - 国产化适配 → 龙芯部署记录 + 技术选型说明（避编译依赖）。
4. **答辩风险预案**：
   - 换例子 → 3~5 个预置案例即点即换；
   - 问未覆盖设备 → "该厂区暂无此设备，但有 X 设备，我演示 X"（老师给的话术）；
   - 追问模拟功能 → "当前为模拟功能，预留第三方 IM 即时通信接口"（老师给的话术）；
   - 现场断网 → 预置内容离线全覆盖。
5. **时间线倒排**（按初赛 7/20）：
   - 本周：龙芯虚拟机到手 → 48 小时内空壳上机；向导第一版 + 案例 1；
   - 下周：后台框架 + 审核回流 + 案例 2；周中开始录初赛视频；
   - 7/18~19：视频定稿与材料提交缓冲。
