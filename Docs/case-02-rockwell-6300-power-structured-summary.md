## 1. 案例一句话

Rockwell ASEM 6300B-EW1 / 6300P-EW1 工控机因设备端 24V DC 供电路径异常、端子松动或供电容量不足，出现不上电、反复重启或异常关机；现场应沿“电源模块输出—设备端电压—Power DC connector—系统启动完整性”逐级排查。

## 2. 设备与故障边界

| 字段 | 内容 |
|---|---|
| 品牌 | Rockwell Automation / Allen-Bradley |
| 设备 | ASEM 6300B-EW1 Box PC、6300P-EW1 Panel PC |
| 场景 | 站控柜、控制室、工业现场 7×24 小时运行 |
| 故障类型 | DC 供电输入异常、容量不足、端子/极性/固定异常、异常断电风险 |
| 典型现象 | Power LED 不亮、无法上电、反复掉电重启、Windows 异常关机、业务软件不稳定 |
| 不属于本案例 | 散热报警、RTC 电池低压、显示故障、网络故障、存储本体故障、主板内部故障 |

## 5. 建议接诊顺序

### 5.1 先确认发生了什么

1. 设备是 6300B-EW1 还是 6300P-EW1；
2. Power LED 是 OFF、绿色常亮、绿色渐变还是绿色闪烁；
3. 是完全不上电、启动中掉电，还是 Windows 启动后重启；
4. 最近是否更换电源、移动端子、进行柜内检修或发生电源波动；
5. POST 是否完成，Windows 是否存在异常关机记录。

### 5.2 再确认发生在哪里

依次测量并比较：

```text
24V DC 电源模块输出
  → 中间断路器 / 端子排
  → Power DC connector
  → Power LED / POST / Windows
```

如果电源模块端正常、设备端异常，优先检查中间断路器、端子排、线缆和 Power DC connector，而不是直接判断主板损坏。

## 6. 原因树

| 一级原因 | 二级原因 | 关键证据 |
|---|---|---|
| 外部电源异常 | 电源模块输出异常、断路器或端子排接触不良 | 上游输出不在 18…32V DC |
| 容量不足 | 未覆盖主机、显示、USB和扩展负载 | 启动或负载变化时电压跌落 |
| DC 接线异常 | DC-松动、极性错误、线径/剥线/扭矩不合规 | 上游正常而设备端压降明显 |
| UPS/关机策略异常 | 未配置 UPS、SYS OFF 策略错误、直接断电 | 异常关机日志、镜像或文件系统异常 |
| 接地/干扰问题 | 多点接地、强干扰设备共线 | 偶发重启或伴随通信异常 |
| 衍生系统故障 | 异常断电造成操作系统或存储损坏 | 供电恢复后仍不能正常启动 |

## 7. 建议检修步骤

1. 通知值班负责人，确认业务影响与停机窗口；
2. 正常关闭 Windows，确认关机流程结束后断开上游电源；
3. 记录 Power LED、POST、接线和原始电压；
4. 测量电源模块输出端和设备 Power DC connector 端电压；
5. 核算电源容量是否覆盖 6300P 85W 或 6300B 60W及扩展负载；
6. 核对 Power DC connector Pin 1 为 +24V、Pin 2 为 0V；
7. 核对 1.5mm² / 16 AWG 导体、7mm剥线长度、端子力矩和应力释放；
8. 恢复供电，观察 Power LED、POST、Windows日志和业务软件；
9. 若供电全部正常但仍无 Power LED 或 POST fatal error，停止一般处理并升级厂家支持。

## 8. 恢复标准

- 设备端电压稳定处于 18…32V DC，正常目标约 24V DC；
- 电源容量覆盖设备与实际扩展负载；
- 极性、线径、剥线长度、端子力矩和应力释放合规；
- Power LED 绿色常亮，POST完成，Windows正常启动；
- HMI/SCADA、数据采集与通信恢复；
- 无新增异常关机事件；
- 若曾发生系统镜像或文件系统损坏，完成修复后才能归档。

## 23. 专家审核候选结果

### 23.1 系统初稿

> 上游24V电源异常导致Rockwell工控机无法启动，恢复电源后设备正常。

该初稿不够准确，因为上游模块在启动瞬间仍为23.9V DC，真正的压降发生在中间端子和设备端之间。

### 23.2 专家修正结论

> 上游24V DC电源模块输出正常，但中间供电路径DC-端子松动且缺少应力释放，导致6300B-EW1设备端启动瞬间由18.4V跌至11.6V，低于官方18…32V DC工作范围。重新制作端头并按手册力矩固定后，设备端恢复24.0V、启动最低23.7V，Power LED绿色常亮，POST、Windows和站控业务软件恢复。

### 23.3 专家新增知识价值

1. “上游24V正常”不能直接证明设备已获得有效供电；
2. 必须比较上游输出、端子排两侧和设备端启动瞬间电压；
3. 额定容量核算与动态压降检查是两个独立步骤；
4. DC端子应同时满足极性、线径、剥线、力矩和应力释放要求；
5. 供电恢复后还必须验证POST、Windows日志和业务软件。
